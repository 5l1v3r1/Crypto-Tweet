# Crypto-Tweet
A python bot that tweets the price and change amount of crypto-currencies.

# Requirements	
Twitter streaming API access.  
tweepy package  
ticker.json file(https://koineks.com/ticker) 

# Setup
1.Clone the repo  
2.pip3 install tweepy  

# Running
python3 bot.py

# Sample bots
BTC: https://twitter.com/bitcoinTLkuru/
